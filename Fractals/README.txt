Привет!

В моей работе реализован весь основной функционал и весь дополнительный,
кроме выбора цвета фрактала. Чтобы нарисовать какой-нибудь фрактал,
нужно выбрать его название из выпадающего списка слева сверху.
Чтобы увеличить размер фрактала, можно использовать выпадающий список
слева снизу. Остальные параметры регулируются с помощью слайдеров.
Чтобы сохранить фрактал, достаточно нажать на кнопку "Сохранить"
и ввести нужную информацию в диалоговом окне. Если нажать на "Сохранить"
до отрисовки фрактала, то будет сохранена пустая область. 
Изображение сохраняется в формате ".png". Для корректного сохранения лучше выбрать
размер фрактала по умолчанию (в остальных случаях может быть сохранена только часть фрактала).

Удачи!

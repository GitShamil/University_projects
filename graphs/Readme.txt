Ввод данных в консоль:
1. Когда программа предлагает несколько вариантов с цифрами, нужно ввести цифру и нажать enter.
2. Имя графа должно состоять из одного слова.
3. Граф следует вводить нужного колличества строк
4. Остальное интуитивно понятно

Работа с файлом.
Файл нужно создавать внутри папки (где лежит этот ридми)
Имя файла нужно вводить так:
pap.txt (к примеру).
Файл должен быть изначально создан.

Устройство самого графа:
Можно вводить только существуеющие графы (если вы ввели граф которого нет, то неопределено). Вершины графа нумеруются с нуля. У каждой вершины графа должно быть хотя бы одно ребро

Матрица смежности
Она вводится как таблица n*n просто -1 0 1. 

Матрица инциденций
Вводится как матрица числа вершин * число ребер.

Adjacency list
в каждую строку вводятся вершины с которыми соотнесена данная. Первая строка это смежные вершины к НУЛЕВОЙ(так как веришны с 0)

List of edges
В каждую строку вводятся два числа 1. вершина из которой идет ребро 
2. вершина куда идет ребро. (ВЕРШИНЫ с 0 нумеруются)
